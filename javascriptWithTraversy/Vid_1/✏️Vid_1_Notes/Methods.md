## JavaScript Crash Course For Beginners

### Topic: Methods

#### SubStrings
   ```javascript

   const s = "hello world"
console.log(s.subScript(0,5));
// result = "hello"
   
   ```
#### Length

```javascript

let s = "hello world"
console.log(s.length);
// result = 11
// space is counted

```

#### toUpperCase

```javascript

let s = "hello world"
console.log(s.toUpperCase);
// result = "HELLO WORLD"

```

#### Split

```javascript

let s = "technology, computers, it, code"
console.log(s.split(","));

```
